📦 Framerly

Framework ultra-ligero sin servidor, sin build, sin compiladores

🔍 ¿Qué es Framerly?

Framerly es un micro-framework que te permite construir aplicaciones SPA sin necesidad de Webpack, Vite ni servidores backend. Funciona sobre rutas declaradas en un JSON y carga HTML dinámicamente en un contenedor < content > aquí < /content >.

📁 Estructura del proyecto


📁 /tu-proyecto
├── index.html
├── router.json
└── templates/
    ├── home.html
    ├── about.html
    └── default.html

🔗 Navegación

Usa el tag personalizado < routes > para generar navegación:


<routes class="flex gap-4">
  <a href="/default">Default</a>
  <a href="/home">Inicio</a>
  <a href="/about">Acerca</a>
</routes>

* Puedes agregar clases como `flex`, `gap`, etc de tailwindcss.

⚙️ router.json

Define tus rutas aquí. Usa `default` como fallback:


{
  "default": "templates/default.html",
  "/home": "templates/home.html",
  "/about": "templates/about.html",
  "/perfil": "templates/perfil.html"
}

📥 Cargar contenido dinámicamente

Puedes usar `data-load="/about"` en cualquier botón o elemento para cargar contenido:


<button data-load="/about" class="bg-blue-600 text-white px-4 py-2 rounded">
  Ir a Acerca
</button>

🎬 Módulo principal

Tu archivo `framerly.js` maneja la navegación, carga de componentes y actualiza la URL con `?page=`.

* También escucha `popstate` para permitir historial con los botones del navegador.

📦 Compatible con Live Server

Sí, puedes correrlo con extensiones tipo "Live Server" en VSCode sin problemas.
