🗪 ¿Qué es Framerly?

Framerly es un micro-framework para crear SPAs sin compiladores ni servidores. Usa rutas JSON y carga HTML dinámicamente de forma declarativa osea que te permite cargar tanto páginas como componentes similar a un framework router real pero sin depender de un servidor, builders ni compiladores todo inline, además guarda el estado y scroll de la página en su última instancia, perfecto para quienes búscan desarrollar landing pages rápidas o SPA simples sin tanta complicacción.

જ⁀➴ Conectarse al CDN

1. Agrega este script al final de tu archivo index.html o de tú página principal que usarás, Este CDN incluye rutas, navegación SPA y carga dinámica de HTML.:

<script type="module" src="https://cdn.jsdelivr.net/gh/JossFramework/framerly/framerly.js"></script>

🗁 Estructura del proyecto

2. Crea esta estructura en tu proyecto, es flexible si adaptas tus rutas puedes crear la estructura que gustes, siempre y cuándo coloques bien las rutas en el router.json:


🗀 /tu-proyecto
├── index.html
├── router.json
└── templates/
    ├── home.html
    ├── about.html
    └── default.html

☀︎ router.json

3. Define aquí tus rutas. Usa default cómo página principal a cargar y components es la ruta donde se encuentran los componentes

{
  "default": "templates/default.html",
  "/home": "templates/home.html",
  "/about": "templates/about.html",
  "components": "templates/components/"
}

✵ Navegación

4. Usa la etiqueta routes para utilizar tus rutas de navegación define varias rutas con la etiqueta <a> </a> : 


<routes>
  <a href="/default">Default</a>
  <a href="/home">Inicio</a>
  <a href="/about">Acerca</a>
</routes>

⌗ Contenido

5. Debes agregar la etiqueta <content> para que el contenido funcione y se cargue puedes colocarla en cualquier lugar de tu página principal ejemplo index.html. 


<content>
  Aquí se cargará las páginas de forma dinámica
</content>

✦ Componentes

Para cargar componentes usa la etiqueta <component src="name"> en src coloca el nombre exacto del componente 


<component src="card">
  Aquí se cargará las páginas de forma dinámica
</component>

⇄ Cargar contenido dinámicamente

También puedes usar data-load="/about" en cualquier botón o enlace:


<button data-load="/about" class="px-4 py-2 bg-gray-800 text-white rounded">
  Ir a Acerca
</button>

⚡ Compatible con Live Server

Framerly funciona perfectamente con extensiones como "Live Server" de VSCode.
